= Редактор разметок

_Редактор разметок_ предназначен для настройки разметок.

.Разметка для просмотра карточек вида "Исходящий", открытая в редакторе разметок
image::outgoing-doc-layout.png[Разметка для просмотра карточек вида Исходящий, открытая в редакторе разметок]

.Области, доступные в окне редактора разметок
include::admin:partial$table.adoc[]
|_Библиотека_
|Коллекция зарегистрированных в {wc}е xref:layouts-general-operations.adoc#layoutsElements[стандартных] и разработанных самостоятельно элементов управления. ЭУ из списка могут быть добавлены в разметку. Элементы управления сгруппированы по назначению: _Системные_, _Согласование_, _Справочники_ и др.

|_Скрипты_
a|Вкладка, на которой выводятся все обработчики событий, используемые в данной разметке. Обработчики событий отображаются в виде раскрывающегося списка со следующей иерархией:

* Верхний элемент списка -- _ЭУ_, для которого назначен обработчик.
** Второй уровень -- _событие_, на которое назначен обработчик.
*** Самый низкий уровень -- _название_ обработчика.

Подробнее см. "xref:controls-event-handler.adoc[]".

|_Структура карточки_
|Древовидное представление структуры разметки с xref:layouts-general-operations.adoc#layoutsRootBlock[корневым блоком] на вершине.

|_Компоновка_
|Упрощенное представление полученной разметки. Блок, соответствующий элементу управления, выбранному в области _Структура карточки_, выделяется красной рамкой.

|_Свойства_
|Свойства элемента управления, выбранного в области _Структура карточки_ или _Компоновка_.
|===

NOTE: Области в окне редактора разметок могут быть отображены или скрыты в зависимости от предпочтений.

.Команды на ленте редактора разметок
include::admin:partial$table.adoc[]
|*Сохранить*
|Сохраняет несохранённые изменения разметки.
|image:buttons/arrow-curve-left.png[Стрелка "отменить"] / image:buttons/arrow-curve-right.png[Стрелка "повторить"]
|Отменяет / повторяет последнее изменение.
|*Отмена*
|Закрывает редактор с запросом на сохранение изменений.
|*Удалить элемент*
|Удаляет выбранный элемент управления с разметки.
|*Вниз* / *Вверх*
|Перемещает выбранный элемент управления по дереву разметки. Элементы управления могут перемещаться в пределах своего родительского блока.
|*Копировать* и *Вставить*
|Копировать или вставить в разметку элемент управления или блок со всем содержимым. При вставке скопированным элементам управления присваиваются новые имена. При копировании сохраняются настройки элементов управления, включая добавленные стили и обработчики событий.
|===

****
Справа от панели команд отображается xref:localizations-default-language.adoc[язык локализации по умолчанию], на котором ведется быстрый поиск по ресурсам локализации.

Если разметка недоступна для редактирования, команды ленты и компоненты в области _Библиотека_ будут скрыты.
