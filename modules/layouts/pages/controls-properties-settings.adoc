= Настройка элементов управления

Настройка элемента управления может включать следующие операции:

. Настройка связи элемента управления с данными карточки.
. Настройка общих свойств элемента управления: текстовой метки, обязательности, видимости и т.д.
. Настройка обработки событий элемента управления.
+

====

Настройка элемента управления выполняется на панели _Свойства_ редактора разметок. Настройка становится доступна после выбора элемента управления в области _Структура карточки_ или в области _Компоновка_ Редактора разметок.

Некоторые свойства элементов управления могут принимать несколько значений.

Локализуемые свойства:: Данные свойства предназначены для локализации разметки. Для каждого языка локализации можно установить собственное значение свойства. В описании элементов управления данные свойства помечены атрибутом `*Локализуемый*`
+
.Локализуемое свойство "Заголовок"
image::properties-header-localization.png[Локализуемое свойство "Заголовок"]
+
Адаптивные свойства:: Данные свойства предназначены для настройки разметки, зависимой от типа пользовательского устройства. Для каждого типа устройств можно установить собственное значение свойства. В описании элементов управления данные свойства помечены атрибутом `*Адаптивный*`
+
.Адаптивное свойство "Видимость"
image::properties-visiblity-for-devices.png[Адаптивное свойство "Видимость"]

Режим редактирования:: Элемент управления, работающий с данными, может иметь данное свойство. Свойство определяет способ изменения значения элемента управления. Подробности в пункте xref:controls-edit-modes.adoc[Настройка режима редактирования].

== CSS стили элементов управления

У элементов управления есть стиль по умолчанию, который определяет, как они будут выглядеть в браузере пользователя. Стандартное оформление элементов управления можно переопределить с помощью собственного стиля (CSS). Подробности в пункте xref:controls-standard-style.adoc[Изменение стандартного стиля элемента управления].

== Как настроить элемент управления

.Чтобы настроить ЭУ:
. Откройте разметку.
. Выберите элемент разметки в области _Структура карточки_ или _Компоновка_. На панели _Свойства_ будут отображены настраиваемые свойства элемента управления.
. В поле `Название` введите уникальное (для данной разметки) название ЭУ. По данному названию, например, можно получить доступ к элементу управления в скрипте карточки.
. Если требуется, настройте `Текст метки`, `Подсказку`, а также другие свойства ЭУ в разделе _Внешний вид_.
. Если ЭУ предназначен для работы с данными (например, карточки), xref:controls-select-data-source.adoc[выберите источник этих данных]. Также можно ограничить возможность изменения значения элемента управления, xref:controls-edit-operation.adoc[указав операцию редактирования].
. Если требуется, настройте свойства в разделе _Поведение_. В частности, здесь можно xref:controls-standard-style.adoc[переопределить стиль] элемента управления.
. xref:controls-event-handler.adoc[Настройте обработку событий элемента управления].

NOTE: Обязательными для настройки являются свойства: `*Источник данных*` и `*Поле данных*`. Остальные свойства имеют значения по умолчанию, которые могут быть сохранены.
