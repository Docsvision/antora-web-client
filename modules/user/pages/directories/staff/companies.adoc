[#createCompany]
== Создать новую организацию

Функции работы со справочником недоступны пока не создана организация. Поэтому для работы со справочником требуется создать как минимум одну организацию.

.Чтобы создать новый узел организации:
. Перейдите в справочник сотрудников.
. В левой области выберите вкладку *Организации*.
+
[NOTE]
====
Допускается создание как самостоятельных организаций, так и создание дочерних организаций.

Организация не будет сохранена, пока не нажата кнопка *Создать*. Это значит, что в процессе создания организации в ней нельзя создавать сотрудников, включая руководителя организации.
====
+
. Выберите необходимый узел.
include::partial$excerpts.adoc[tags=createMethods]
. Заполните все необходимые поля и нажмите *Создать*.
+
Подробнее о полях карточки организации см. <<companyFields,Перечень полей карточки организации>>.

== Изменить организацию

.Чтобы изменить созданный ранее узел организации:
. На вкладке _Организации_ в левой области справочника выберите узел из иерархического списка.
include::partial$excerpts.adoc[tags=editMethods]
. Измените необходимые <<companyFields,поля>> и нажмите *Сохранить*.

== Удалить организацию

.Чтобы удалить созданный ранее узел организации:
. На вкладке _Организации_ в левой области справочника выберите узел из иерархического списка.
. Вызовите контекстное меню на выбранном узле.
. Выберите соответствующий пункт из контекстного меню, чтобы удалить узел.
. Появится предупреждение, подтвердите действие кнопкой *ОК*.

|Контактное лицо
|В поле указывается сотрудник, который является официальным представителем организации. Контактное лицо, как и руководителя, можно выбрать из справочника или при помощи быстрого поиска.

|Адрес
|В поле <<address,указывается>> фактический адрес. При изменении адреса в поле `Адрес` в разметке создания или изменения организации, информация будет скопирована в поля `Почтовый адрес` и `Юридический адрес`.

|Почтовый адрес
|В поле <<address,указывается>> адрес, который будет использован для отправки почтовой корреспонденции.

|Юридический адрес
|В поле <<address,указывается>> адрес регистрации организации.

[#address]
=== Адрес

При выборе поля `Адрес`, `Почтовый адрес` или `Юридический адрес` из карточки организации открывается новое окно для добавления адреса:

.Окно добавления адреса организации
image::create-new-company-address.png[Окно добавления адреса организации]

[NOTE]
====
При изменении адреса в поле `Адрес` в разметке создания или изменения организации, информация будет скопирована в поля `Почтовый адрес` и `Юридический адрес`.
====

При добавлении адреса для заполнения доступны следующие поля:

[horizontal]
Индекс:: Почтовый индекс организации.
Город:: Город, в котором организация находится.
Адрес:: Улица, дом, офис.
Страна:: Страна нахождения организации.
