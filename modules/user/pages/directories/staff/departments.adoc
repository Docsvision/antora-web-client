= Работа с подразделениями

В справочнике сотрудников реализована возможность создания подразделений организаций. Каждая организация может иметь неограниченное количество подразделений. Подразделения являются составной частью организаций и не могут быть созданы вне организации.

[#info]
== Посмотреть информацию о подразделении

Основная информация о выбранной организации или подразделении доступна в правой области справочника, во вкладке _Информация_, полную информацию можно посмотреть, выбрав пункт _Информация_ в контекстном меню.

. На вкладке *Организации* в левой области справочника выберите узел из иерархического списка.
. Выберите Подразделение.
include::partial$excerpts.adoc[tags=editMethods]

[#delete]
== Удалить подразделение

.Чтобы удалить ранее созданный узел подразделения:
. На вкладке _Организации_ в левой области справочника выберите узел из иерархического списка.
. Вызовите контекстное меню на выбранном узле.
. Выберите соответствующий пункт из контекстного меню, чтобы удалить узел.
. Появится предупреждение, подтвердите действие кнопкой *ОК*.

[#fields]
== Перечень полей карточки подразделения

Основная информация о подразделении может быть заполнена при создании нового подразделения или изменена после создания.

.Основная информация о подразделении
// image::create-new-dept-main.png[Основная информация о подразделении]
image::create-new-dept-main-index.png[Основная информация о подразделении]

Название::
В поле указывается сокращенное название, которое будет отображаться в справочнике {dv}.
Полное название::
В поле указывается полное официальное название.
Индекс подразделения::
В поле указывается индекс подразделения.
Руководитель::
В поле указывается руководящий сотрудник.

Контактное лицо::
В поле указывается сотрудник, который является официальным представителем подразделения. Контактное лицо можно выбрать из справочника или при помощи быстрого поиска, как и руководителя.
E-mail::
В поле указывается адрес для отправки электронных писем.
Телефон::
В поле указывается номер телефона для связи.
Адрес::
В поле xref:directories/staff/companies.adoc#address[указывается] фактический адрес. При изменении адреса в поле `Адрес` в разметке создания или изменения подразделения, информация будет скопирована в поле `Почтовый адрес`.
Почтовый адрес::
В поле xref:directories/staff/companies.adoc#address[указывается] адрес, который будет использован для отправки почтовой корреспонденции.

[NOTE]
====
Поля `Адрес` и `Почтовый адрес` заполняются аналогично с xref:directories/staff/companies.adoc#address[полем] `Адрес` в данных организации.
====
