= Включить аутентификацию Kerberos

Чтобы иметь возможность использования сквозной аутентификации, когда сервер расположен в ОС Linux, необходимо выполнить специальные настройки Kerberos и используемого браузера.footnote:[Пример настроек можно запросить через службу технической поддержки {dv}.]

.Обязательные условия для настройки сквозной аутентификации:
* Обращение к {wc}у по доменному имени и адресу `\http://example.domain.com:5004/Account/LoginWindows`.
* Сервер {wc}а должен находиться в домене.
