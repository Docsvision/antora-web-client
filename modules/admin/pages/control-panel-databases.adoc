= Раздел "Базы данных"

== Настройки подключения к серверу

Раздел _Базы данных_ открывает страницу настроек подключения {wc} к серверу {dv}.

.Страница настроек "Базы данных"
image::control-panel-db.png[Страница настроек "Базы данных"]

.Страница настроек "Базы данных"
include::partial$table.adoc[]
|`Адрес сервера`
|Полный или сокращенный адрес сервера платформы {dv}, к которому подключается {wc}.

// |image:buttons/save.png[save]
// |Сохраняет адрес сервера после изменения.

|_Тенанты_
|Список подключенных к {wc} баз данных указанного сервера {dv}.
|===

NOTE: Если нужная БД в списке отсутствует, её можно добавить через _{cns}_ при помощи xref:6.1@platform:console:db-connect.adoc[Мастера баз данных].

[NOTE]
====
После добавления новой БД необходимо её настроить. Потребуется проверить локализации, импортировать решения и настроить виды карточек по умолчанию.
Подробнее см. п. xref:settings-extended.adoc[Расширенная настройка].
====

== Изменение адреса подключения к серверу

При необходимости адрес сервера {dv} может быть изменён.

В поле `Адрес сервера` указывается адрес подключения к серверу {dv}.
//+
// ****
// include::partial$rest-soap.adoc[tags=part1]
// ****
//+
//. Нажмите на кнопку image:buttons/save.png[Сохранить].
//+
//Если был указан адрес формата SOAP, он будет сокращен до `\http://company.com/{dv}/`. Если в конфигурационном файле {wc} xref:change-transport.adoc[указан транспорт COM], сокращение адреса не выполняется.
//
Пользователи Модуля могут работать с xref:multi-tenant-mode.adoc[несколькими БД]. Текущей активной считается БД, псевдоним которой указан в адресной строке до первой точки, например: `\http://{dv}DB.company.com:5004`.
